# QED - a simple command line editor

## Installation
1. Clone this repository
2. run `make`

## Usage
Run `qed <filename>`. It will open the file or create it if it does not exist

## Commands

### Add or edit line
```
<line-number> <line-content> 
```
Example - `5 hey there`
Line numbers start from `0`

### Show current texts
```
s
```

### Quit and save
```
quit
```
